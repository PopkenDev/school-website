ESP8266 based smart alarm clock

Introduction

The ESP8266-based smart alarm clock project is a comprehensive endeavor aimed at resolving multiple issues related to household lighting, alarm functionality, and weather information. This blog post will delve into the project's objectives, components, circuit design, and software aspects.

Objectives

  • Shifting lighting control from light switches to smartphone
  • Enhanced functionality with a more sophisticated device with additional features.
  • Display the time in a user-friendly manner.
  • Wake the user up gently using sounds and a simulated sunrise.
  • Provide weather forecast information for better planning.
  • Incorporate adjustable display brightness for nighttime use.
  • Enable remote control of bedroom lights via MQTT.
  • Offer easy configuration through a web interface.
  • Automatically synchronize time using NTP.
  • Ensure an aesthetically pleasing design for household acceptance.

Components

  • Wemos D1 Mini (ESP8266-based microcontroller).
  • 0.96" Dual-color OLED display for weather forecasts.
  • Blue 7-segment clock display with I2C interface.
  • Liquid Crystal Light Valve for adjustable display dimming.
  • TIP120 MOSFET transistors for RGB+W LED strip control.
  • RGB+W LED Strip (reused from a previous project).
  • Blue ring 3.3V push button for snooze and smart switch.
  • Three small push buttons for manual alarm time setting.
  • Assorted resistors for voltage dividers (ranging from 10K to 1K).
  • A 20K potentiometer with a knob.
  • 12V / 1.5A power supply (repurposed from a USB hard drive).
  • DC power jack for power input.

Circuit Diagram

  • Power Supply: A 12V/1.5A input is converted to 5V using a voltage regulator to power the Wemos D1 Mini. It also supplies power to the RGB+W LED strip.
  • LED Strip Control: The circuit regulates the intensity of each color in the RGB+W LED strip based on commands received from the Wemos D1 Mini via GPIOs.
  • Video Bus: This block manages the I2C bus that connects both the OLED and 7-segment displays.
  • Alarm and Snooze: Includes a buzzer for alarm functionality and a snooze button to stop the alarm.
  • Alarm Time Setting: Three buttons allow manual adjustment of the alarm time.
  • Controllable Shutter Glass: A potentiometer controls the translucency of the liquid crystal light valve.

Software

The project's software can be found in the GitHub repository, specifically designed for the ESP8266 Arduino IDE. It's essential to use the latest version (2.4.2) to avoid issues with the "tone" and "analogWrite" functions, as earlier versions exhibited undesirable behavior.

Web Interface

A user-friendly web interface developed for configuring the alarm clock, featuring tabs for alarm settings, timezone selection, MQTT setup, Weather Underground API integration, and manual control of LED backlighting.
